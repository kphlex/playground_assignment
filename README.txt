When user visits localhost:XXXX/play renders 3 blue boxes.

When user visits localhost:XXXX/play/(x) it renders the amount of blue boxes user
entered into x. Ex - localhost:5000/play/10, will render 10 blue boxes. 

When user visits localhost:XXXX/play/(x)/(color) it renders the amount of boxes 
and the color the user entered. Ex - localhost:5000/play/10/black, will render 10 black
boxes.

Resposive to various screen sizes.
